# Day 2 - Multi Page Website

## Overview

This project is the second milestone in my UI/UX Design and Full Stack Development learning journey. The goal of this project was to understand how real websites are structured using multiple pages and how navigation links connect different sections of a website.

The website consists of three pages:

* Home Page
* About Page
* Contact Page

Each page is connected through a navigation menu, allowing users to move easily between sections.

## Features

* Multi-page website structure
* Navigation menu linking all pages
* About Me section
* Skills section
* Contact information section
* Responsive design
* Clean and organized layout
* External CSS styling

## Technologies Used

* HTML5
* CSS3

## What I Learned

Through this project, I learned:

* How multi-page websites work
* Creating and linking multiple HTML files
* Using navigation menus
* Organizing website content
* Applying consistent styling across pages
* Building responsive layouts with CSS

## Project Structure

day-2-multi-page-website/

├── index.html

├── about.html

├── contact.html

└── style.css

## Future Improvements

* Add JavaScript interactivity
* Improve UI design
* Add animations and transitions
* Create a responsive navigation menu
* Build more advanced multi-page websites
